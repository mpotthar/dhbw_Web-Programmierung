Über dieses Projekt

Willkommen auf der Informationsseite des Autobahn-Informationsportals. Hier finden Sie Antworten auf häufig gestellte Fragen zu diesem Projekt.

Die auf dieser Website angezeigten Daten stammen von der offiziellen Autobahn App API. Diese API wird von der Autobahn GmbH des Bundes bereitgestellt und bietet aktuelle Informationen über das deutsche Autobahnnetz.

Für dieses Projekt wurden zwei spezifische Endpoints der Autobahn API genutzt:
  1. Der Endpoint für die Liste aller Autobahnen: https://verkehr.autobahn.de/o/autobahn/
  2. Der Endpoint für Baustelleninformationen einer spezifischen Autobahn: https://verkehr.autobahn.de/o/autobahn/{autobahn_id}/services/roadworks
Diese Endpoints liefern die notwendigen Daten für die Darstellung der Autobahnen und ihrer aktuellen Baustellen nebst Baustellendetails.

Bootstrap wurde für diese Website aus mehreren Gründen gewählt:
  • Responsive Design: Bootstrap ermöglicht eine einfache Erstellung von mobilfreundlichen, responsiven Layouts.
  • Konsistentes Erscheinungsbild: Es bietet vordefinierte Stile und Komponenten für ein einheitliches Design.
  • Schnelle Entwicklung: Mit Bootstrap ist es möglich, schnell ein professionelles Layout zu erstellen.
  • Große Community: Es gibt eine umfangreiche Dokumentation und viele Ressourcen für Entwickler.
  • Kompatibilität: Bootstrap funktioniert gut mit verschiedenen Browsern und Geräten.

Der Hauptnutzen unserer Website besteht darin, Autofahrern und anderen Interessierten einen schnellen und einfachen Überblick über aktuelle Baustellen auf deutschen Autobahnen zu geben. Nutzer können:
  • spezifische Autobahnen auswählen und deren Baustellen einsehen
  • detaillierte Informationen zu jeder Baustelle abrufen
  • die Lage der Baustellen auf interaktiven Karten visualisieren
  • ihre Reisen besser planen und potenzielle Verzögerungen antizipieren
Ziel dieser Website ist es, die Reiseplanung zu erleichtern und die Sicherheit auf deutschen Autobahnen zu erhöhen.

Die Daten auf dieser Website werden in Echtzeit von der Autobahn App API abgerufen. Jedes Mal, wenn ein Nutzer eine spezifische Autobahn auswählt oder die Seite aktualisiert, werden die neuesten verfügbaren Informationen von der API angefordert. Die Häufigkeit der Aktualisierungen der API selbst hängt von der Autobahn GmbH des Bundes ab.

Die Kartenintegration wurde mit Leaflet.js realisiert, einer Open-Source-JavaScript-Bibliothek für interaktive Karten. OpenStreetMap wurde als Kartendatenquelle verwendet. Für jede Baustelle wird eine individuelle Karte erstellt, die den genauen Standort und, falls verfügbar, die Geometrie der Baustelle anzeigt. Die Hauptkarte zeigt eine Übersicht aller Baustellen auf der ausgewählten Autobahn.

Die auf dieser Website dargestellten Informationen basieren auf Daten der Autobahn App API. Trotz sorgfältiger Verarbeitung kann keine Gewähr für die Richtigkeit, Vollständigkeit und Aktualität der angezeigten Daten übernommen werden. Die Daten werden von einer externen Quelle bezogen und können möglicherweise nicht den aktuellsten Stand wiedergeben. Auch ist es möglich, dass Daten nicht oder mehrfach über die API ausgeliefert werden, da entsprechende Datensätze nicht bzw. redundant vorhanden sind. Dies betrifft auch die Auflistung der Autobahnen. Eine Anzeige von N/A im Abschnitt deutet darauf hin, dass seitens der API keine Daten vorliegen. Dies ist normalerweise dann der Fall, wenn sich die Baustelle in einer Auf- bzw. Abfahrt befindet.
